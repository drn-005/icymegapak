# ICY MEGA PAK
A bundle of 2 batch file "paks" that contain various boredom killers, such as literature, a movie, music, and some games.

Hey there! This is a tutorial for using the Icy Mega Pak. First, let's talk about the IcyCoolPak, the main asset downloader.

## <<\<ICYCOOLPAK>>>
Upon executing the IcyCoolPak.bat file, a cyan and black command window will pop up and tell you a few things. Don't worry, this is normal. You just have to wait for the BITS Transfer to complete, which will take around 10-20 minutes depending on your internet speed.

## <<\<EXTRACTION>>>
Once the download finishes, you'll want to look at your desktop and open the new folder called IcyCoolPak. Inside are 3 .zip files. You need to extract these 3 files through your file explorer. Once all 3 are extracted, there should be 6 things in IcyCoolPak, those being the .zip files you just extracted and 3 more new folders with the names Games, Literature, and Media.

## <<\<RETROARCH>>>
Inside the Games folder, you'll find an app called "RetroArch-Win64-setup.exe". Open it and follow the setup guide. Once you're in RetroArch, navigate to the Main Menu > Online Updater > Core Downloader, then download these cores:
* Arcade (MAME 2003-Plus)
* Nintendo - Game Boy/Color (Gambatte)
* Nintendo - Game Boy Advance (mGBA)
* Nintendo - NES/Famicom (QuickNES)
That's all you need to do in RetroArch, so close it out. Make sure NOT to unzip the 5 game files in the Games folder.

## <<\<ICYHAXXPAK>>>
Simply run the IcyHaxxPak.bat file and navigate through the menus to get to your entertainment of choice.

That's all you need to know about Icy Mega Pak!
